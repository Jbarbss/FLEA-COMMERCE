## Description

  This E-Commerce website displays products for purchase. Users need to create an account and log in to post items to their dashboard. They can then add/remove/update items as they see fit.

## Installation

- You must install Both Node.js and MySQL on your computer.
- Clone the repo by copying and pasting in your command line: 
- `git clone git@github.com:/Jbarbss/Project-2`
- Navigate to the root directory and run: 
- `npm install`
- Navigate to the root directory: 
- Add a .env file to the root of the project
- Navigate to the seeds directory: 
- Add a .env file to the seeds folder
- To start the server, in the command line run: 
- `npm start`
